## Installation

Clone the project

```bash
  git clone git@github.com:CatherineJOnia/Portfolio.git
```

Enter the project

```bash
  cd Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Access the project in your browser at [`localhost:3000`](http://localhost:3000)
